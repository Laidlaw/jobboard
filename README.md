jobboard
========

A prototype for a jobboard built in angular

## Pages

- Public (not logged in)
  - Home Page (sign up page like tumblr)
    - Sign Up
    - Log in
    - window shop
  - Search Page
  - Jobs Page (results)
    - additional filters
  - Event page
    - explanation
    - call to action

- Seeker Logged In
- Job Seeker Dashboard (home page if signed in)
